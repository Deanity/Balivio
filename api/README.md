# Balivio Backend API

RESTful API server for Balivio, a Bali villa search and booking platform.

## Architecture

This project is built using:
- **Express.js v5** (routing and HTTP controllers)
- **TypeScript** (strict mode)
- **Drizzle ORM** (type-safe database queries and migrations)
- **Supabase (PostgreSQL)** (hosted database)
- **Zod** (strict request validation)
- **bcryptjs** & **jsonwebtoken** (custom password hashing and JWT sessions)
- **Xendit SDK** (payment gateway integration)
- **Vitest** & **Supertest** (testing)

### Project Layout

```
api/
  src/
    config/          # Env variables, Supabase, DB configs
    db/
      schema/        # Drizzle schema definitions
      migrations/    # DB migrations auto-generated by drizzle-kit
      seed/          # DB seeding scripts
    middleware/      # Auth validation and rate-limit middlewares
    modules/         # Domain layers
      router/        # Routes definitions
      controller/    # Request parse and validation handling
      service/       # Main core business logic and DB operations
      schema/        # Zod payload structures
    utils/           # Pagination, response helpers, JWT and hashing
    types/           # custom express/api interfaces
    app.ts           # Express application setup
    server.ts        # Server entrypoint
```

## Setup & Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Setup Environment Variables:
   Copy `.env.example` to `.env` and fill in the values:
   ```bash
   cp .env.example .env
   ```

3. Database Operations:
   - Generate database migrations from schemas:
     ```bash
     npm run db:generate
     ```
   - Apply migrations to Supabase:
     ```bash
     npm run db:migrate
     ```
   - Seed database:
     ```bash
     npm run db:seed
     ```

4. Start Development Server:
   ```bash
   npm run dev
   ```
